# Task Manager Console Application
## Description
A console-based Task Manager developed using Python.
## Features
- Add Task
- View Tasks
- Update Task Status
- Delete Task
- Menu Driven Interface
## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Lists
- CRUD Operations
- Functions
- Loops and Conditionals
